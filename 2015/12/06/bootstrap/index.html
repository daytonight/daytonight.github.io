bootstrap简介：

bootstrap是由Twitter开发的一个开源框架，是一套具有易用、优雅、灵活、可扩展等特性的前端工具集，它提供的用户组界面组件可用于创建整洁且具有吸引力的网页，而且这些网页还能兼容所有现代Web浏览器。
boostrap语法：

第一章  排版

  • 标题：
    < hn >:n级标题
    < small >:副标题
  • 段落：
    < p >:段落
    class=”lead”*–段落突出显示（增大文本字号，加粗文本）
  • 加粗
    < b >(< strong >)

  • 斜体
    < em >(< i >)

  • 强调相关的类(可在< div >容器或< p >等的class类名中定义)
    text-muted：提示，使用浅灰色（#999）

    text-primary：主要，使用蓝色（#428bca）

    text-success：成功，使用浅绿色(#3c763d)

    text-info：通知信息，使用浅蓝色（#31708f）

    text-warning：警告，使用黄色（#8a6d3b）

    text-danger：危险，使用褐色（#a94442）

  • 文本对齐方式（class类名）：
    text-left：左对齐
    text-center：居中对齐
    text-right：右对齐
    text-justify：两端对齐

  • 定义列表：
    < dl >
    　　< dt >定义列表标题< /dt >
    　　< dd >定义列表项< /dd >
    < /dl >

  • 水平定义列表(标题和列表项水平显示)：< dl class=”dl-horizontal”>
  • 去点列表：class=”list-unstyled”

  • 内联列表：class=”list-inline”

  • 代码：
    < code >:单个单词或句子的代码
    < pre >:多行代码（成块的代码）
    < pre class=”pre-scrollable” >:为多行代码加滚动条
    < kbd >:用户输入的代码（黑体显示）

  • 表格： < table class=” “>
    class=
    table:基础表格
    table-strip:斑马线表格
    table-bordered:带边框的表格
    table-hover：鼠标悬停高亮的表格
    table-condensed:紧凑型表格（单元格内距较小）
    table-responsive：响应式表格

  • 将表格置于< div class=”table-responsive” >容器中即可

表格行的类：< tr class=” “>
class=
active： 表示当前活动的信息
success ：表示成功或者正确的行为
info： 表示中立的信息或行为
warning: 表示警告，需要特别注意
danger: 表示危险或者可能是错误的行为

第二章  表单

  • 水平表单：

    1. 在< form >元素中使用“form-horizontal”类名
    2. 配合Bootstrap框架的网格系统

  • 内联表单（表单的元素均在一行内显示）：< form class=”form-inline” >

  • 表单控件：在控件中使用”form-control”类名后，将会实现一些设计上的定制效果

  • 输入框：< input type=” “ class=”form-control” >

  • 下拉选择框：< select multiple class=”form-control” >
  • 文本域： < textarea class=”form-control” rows=”3” >由于设置了该class类名，文本域宽度默认为auto只需设置高度rows

  • 复选框checkbox和单选按钮radio:< input type=”checkbox” value=”” >、< input type=”radio” name=” “ id=” “ value=” “>

    • 如果checkbox需要水平排列，只需要在label标签上添加类名“checkbox-inline”
    • 如果radio需要水平排列，只需要在label标签上添加类名“radio-inline”(< label >为按钮的容器)

  • 按钮
    input[type=“submit”]：提交按钮
    input[type=“button”]：普通按钮
    input[type=“reset”]：重置按钮
    < button class=”btn btn-…”  type=”button “ >
    < span class=”btn btn-…”> ：span标签按钮
    < div class=”btn btn-…”>:div按钮
    < a href=”##” class=”btn btn-…” >:a按钮

    • bootstrap通常采用< button >按钮和< a >按钮，通过设置btn类名可设置按钮的种类（不同颜色）：
      “btn”:基础按钮
      “btn btn-default”:默认按钮
      “btn btn-primary”：主要按钮
      “btn btn-success”成功按钮
      “btn btn-warning”：警告按钮
      “btn btn-danger”：危险按钮
      “btn btn-link” :连接按钮
      “btn btn-info”：信息按钮

  • 按钮大小：

    1. < button class=”btn btn-primary btn-lg” type=”button” >：大型按钮btn-lg
    2. < button class=”btn btn-primary btn-sm” type=”button” >：小型按钮btn-sm
    3. < button class=”btn btn-warning btn-xs” type=”button” >:超小型按钮btn-xs

    4. < button class=”btn btn-primary” type=”button” >:正常按钮

    5. “btn btn-block”:块状按钮（按钮充满整个父容器）

  • 按钮状态（禁用）：
    方法1：在标签中添加disabled属性（disabled=”disabled”）
    方法2：在元素标签中添加类名“disabled”

  • 图像：< img class=”img-…” alt=”140x140” src=” “>
    class=”img-responsive”：响应式图片，主要针对于响应式设计
    class=”img-rounded”：圆角图片
    class=”img-circle”：圆形图片
    class=”img-thumbnail”：缩略图片
