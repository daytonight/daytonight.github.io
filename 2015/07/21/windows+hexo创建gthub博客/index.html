windows+hexo创建github博客

hexo环境下创建github博客

如何搭建hexo环境

下载git

下载对应系统的node.js安装包并安装

手动创建工作目录hexo运行如下代码

hexo init
hexo server

此时本地服务器已能看到静态网页

复制好看的主题

参见如何搭建一个独立博客

复制完后需要更改_config.yml文件，最好的方法是直接复制过来，然后执行

cd jacman
git pull

将本地静态网页写到github上

编辑—config.yml

deploy:
  type: github
  repository: your URL
  branch: master

执行下列指令完成部署

hexo g
hexo d

以上步骤执行完后会要求输入用户名和密码，完成输入后进入：
username.github.io即能看到新生成的博客页面

新用户可重新设置ssh

此后可执行以下代码以向博客中添加文章

hexo new
hexo g
hexo d

常用指令

  • hexo init:用git bash打开执行后自动生成建站所需所有文件

  • npm install：与上一条指令联合使用

  • npm instll hexo-deployer-git —save:一般hexo deploy出错的时候要执行这条指令，在这里提前安装好

  • hexo g:生成文章
  • hexo s:启动本地服务，预览前面提到的静态网页
  • hexo d:部署
