linux+hexo创建github博客

#linux+hexo创建github博客

##本次博客从windows上迁移到linux上大概分为以下几步：

  • 安装linux并熟悉基本擦作：
  1. 目录处理命令
  2. 文件处理命令
  3. 权限管理命令
  4. vim和gedit的使用

  • 我这里是用虚拟机安装的centos7

  • 搭建hexo环境

  1. 通过yum源安装git

    sudo yum install git-core

  2. 安装nvm

    git clone https://github.com/cnpm/nvm.git

./install.sh
source /home/git/nvm.sh

  3. 通过 nvm 安装node

    nvm install v0.12.5
nvm use v0.12.5

  4. 通过nvm安装hexo

    npm install -g hexo-cli
hexo init "your hexo filename"
npm install

##将windows上创建的博客迁移到linux下（在不同电脑下用hexo更新github博客）

  1. 下载github for windows
  2. 将存放博客内容的分支克隆到本地
  3. 将.html博客内容转换成.md
  4. 复制博客内容到linux下重新生成和部署
  5. 新建博客并发布到github上(方法和windows下相同)
    #总结和感悟
    关于centos的版本，我安装过cento6.5和centos7以及basic server和desktop，最后决定还是安装最新版本的好点。对于linux的熟悉部分目前仍还在进行，这也是本次写博客面临的第一个大问题。linux下的gedit编辑器真是好用，之前还特意去找了linux下的能实时预览的markdown，然后写了一下才发现gedit差不多就能预览了，还支持各种格式。此次博客最后一步的迁移虽然方法有点白痴（是我自己想的），不过是目前我能做到的最简单的方法了。
